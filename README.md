# React + TypeScript + Vite

This is a [React.js](https://react.dev/) project bootstrapped with [Vite](https://vitejs.dev/)

## Getting Started

```bash
npm run dev
# or
yarn dev
# or
#pnpm dev
# or
bun dev
```
Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.
